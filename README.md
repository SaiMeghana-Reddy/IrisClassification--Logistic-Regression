# 🌼 Iris Flower Classification using Logistic Regression

## 📖 Project Overview
This project builds a Machine Learning model to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

The model is implemented using Logistic Regression, along with data visualization.

---

## 🎯 Objective
To apply Logistic Regression on the Iris dataset and analyze its performance in classifying flower species.

---

## 📊 Dataset
- File: `Iris.csv`
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Species

---

## 🔍 Exploratory Data Analysis (EDA)
The project includes:
- Scatter plots for feature comparison
- Basic insights into data patterns

---

## ⚙️ Model Used
- Logistic Regression
